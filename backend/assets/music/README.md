# Music Assets Directory

Tempat menyimpan file musik royalty-free untuk background music.

## Built-in Tracks yang Diperlukan:

1. `happy_upbeat.mp3` - Musik ceria untuk konten fun
2. `chill_lofi.mp3` - Lofi beats untuk konten santai
3. `dramatic_cinematic.mp3` - Musik dramatis untuk konten serius
4. `energetic_rock.mp3` - Musik energik untuk konten action
5. `inspiring_piano.mp3` - Piano inspiratif untuk konten motivasi

## Catatan:
- Pastikan semua file musik adalah royalty-free
- Format: MP3, 128-320kbps
- Durasi minimal: 60 detik (akan di-loop otomatis jika perlu)
